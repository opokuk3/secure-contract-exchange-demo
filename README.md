# Secure Contract Exchange Demo

## Overview
This project is a Flask web application that demonstrates a secure contract exchange protocol using real cryptographic operations. It simulates a multi-step workflow in the browser, including certificate generation, trust establishment, key exchange, encryption, digital signatures, verification, and tamper detection.

## Features
- Interactive browser-based protocol walkthrough
- X.509 certificate generation
- ECDH shared secret derivation
- AES-256-GCM authenticated encryption
- ECDSA digital signatures and verification
- Tamper detection through hash/signature mismatch
- Live display of cryptographic values during the simulation

## Technologies Used
- Python
- Flask
- cryptography
- HTML / CSS / JavaScript
